# Angular CDK Sample

This app shows some of the features of the [Angular Component DevKit (CDK)](https://material.angular.io/cdk/)

It shows the following features:
* Platform utils
* Layout observers
* Tree component
* Table component
* Overlays
* Drag and Drop
* Virtual Scrolling
